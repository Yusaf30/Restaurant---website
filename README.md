# Spice Heaven Restaurant Website 🍽️

This is a simple restaurant website project built using **HTML**.
The website represents a fictional restaurant called **Spice Heaven Restaurant** and includes multiple sections such as menu, popular dishes, customer reviews, reservation, and contact information.

The goal of this project is to practice **basic HTML structure, semantic elements, and multi-page website design**.

---

## 📌 Features

* Home page with restaurant introduction
* Popular dishes section
* Customer reviews section
* Restaurant menu page
* Food gallery page
* Table reservation page
* Contact information page
* Social media follow section

---

## 🛠️ Technologies Used

* HTML5
